## Usage

To run the script, use the following command in your terminal:

```bash
python main.py <algorithm> <input_file>
```
where *algorithm* is one of following:
  - `DFS`
  - `BFS`
  - `A1`
  - `A2`

and input file is located in directory *inputs*
